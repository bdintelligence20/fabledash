  <name>Task 1: Create foundational UI components (Button, Card, Input, Select, Spinner, Avatar, Badge)</name>
  <files>src/components/ui/Button.tsx, src/components/ui/Card.tsx, src/components/ui/Input.tsx, src/components/ui/Select.tsx, src/components/ui/Spinner.tsx, src/components/ui/Avatar.tsx, src/components/ui/Badge.tsx</files>
  <action>
Create each component in src/components/ui/ with the following specifications. Use `clsx` or simple template literal class merging (do NOT add clsx as a dependency — use a tiny local `cn` helper or inline template literals). All components accept `className` prop for override/extension.

**Button.tsx:**
- Props: `variant` ('primary' | 'secondary' | 'ghost' | 'danger'), `size` ('sm' | 'md' | 'lg'), `loading` (boolean), `icon` (ReactNode, optional leading icon), `children`, plus all native button props
- Variants: primary uses `bg-primary-600 hover:bg-primary-700 text-white`, secondary uses `bg-white border border-surface-200 text-surface-700 hover:bg-surface-50`, ghost uses `text-surface-600 hover:bg-surface-100`, danger uses `bg-danger-600 hover:bg-danger-700 text-white`
- Sizes: sm (px-3 py-1.5 text-sm), md (px-4 py-2 text-sm), lg (px-5 py-2.5 text-base)
- All: `rounded-lg font-medium transition-default focus:ring-focus disabled:opacity-50 disabled:cursor-not-allowed`
- When `loading`, show Spinner inline and disable button
- Use `forwardRef` for ref forwarding

**Card.tsx:**
- Props: `padding` ('none' | 'sm' | 'md' | 'lg'), `hover` (boolean — adds hover shadow), `children`, plus native div props
- Base: `surface-card` utility class from index.css
- Padding: none (p-0), sm (p-4), md (p-6), lg (p-8)
- Hover: adds `hover:shadow-card-hover transition-default cursor-pointer`
- Export sub-components: `Card.Header`, `Card.Body`, `Card.Footer` as simple styled divs with appropriate padding and borders

**Input.tsx:**
- Props: `label` (string, optional), `error` (string, optional), `icon` (ReactNode, optional leading icon), plus all native input props
- Wrapper div with label above, input, error message below
- Input: `w-full rounded-lg border border-surface-200 bg-white px-4 py-2.5 text-sm text-surface-900 placeholder:text-surface-400 focus:border-primary-500 focus:ring-focus transition-default`
- Error state: border-danger-500, error text in text-danger-600 text-sm
- With icon: icon positioned absolute left, input gets pl-10
- Use `forwardRef`

**Select.tsx:**
- Props: `label` (string, optional), `error` (string, optional), `options` ({value: string, label: string}[]), `placeholder` (string), plus native select props
- Same styling as Input for consistency
- Use `forwardRef`

**Spinner.tsx:**
- Props: `size` ('sm' | 'md' | 'lg'), `className`
- Simple animated SVG spinner (circular) using `animate-spin`
- Sizes: sm (h-4 w-4), md (h-6 w-6), lg (h-8 w-8)
- Default color: `text-primary-600`, overridable via className

**Avatar.tsx:**
- Props: `name` (string), `src` (string, optional image URL), `size` ('sm' | 'md' | 'lg'), `className`
- Shows image if `src` provided, otherwise shows initials (first letter of first two words)
- Sizes: sm (h-8 w-8 text-xs), md (h-10 w-10 text-sm), lg (h-12 w-12 text-base)
- Initials variant: `bg-primary-100 text-primary-600 font-semibold`
- Rounded-full always

**Badge.tsx:**
- Props: `variant` ('default' | 'primary' | 'success' | 'warning' | 'danger'), `size` ('sm' | 'md'), `dot` (boolean — shows colored dot before text), `children`
- Variants: default (bg-surface-100 text-surface-700), primary (bg-primary-50 text-primary-700), success (bg-success-50 text-success-700), warning (bg-warning-50 text-warning-700), danger (bg-danger-50 text-danger-700)
- Sizes: sm (px-2 py-0.5 text-xs), md (px-2.5 py-1 text-sm)
- All: `inline-flex items-center rounded-full font-medium`

  <name>Task 2: Create complex UI components (Modal, Table, StatCard, Tabs) and barrel export</name>
  <files>src/components/ui/Modal.tsx, src/components/ui/Table.tsx, src/components/ui/StatCard.tsx, src/components/ui/Tabs.tsx, src/components/ui/index.ts</files>
  <action>
**Modal.tsx:**
- Props: `isOpen` (boolean), `onClose` (() => void), `title` (string), `size` ('sm' | 'md' | 'lg' | 'xl'), `children`
- Uses React Portal (createPortal to document.body)
- Backdrop: `fixed inset-0 bg-black/50 z-40` with click-to-close
- Panel: centered, `bg-white rounded-2xl shadow-strong z-50` with slide-up animation
- Sizes: sm (max-w-sm), md (max-w-md), lg (max-w-lg), xl (max-w-xl)
- Header: title + close button (X icon from lucide-react)
- Body: overflow-y-auto with max-height
- Close on Escape key (useEffect with keydown listener)
- Animate in with `animate-slide-up`

**Table.tsx:**
- Export `Table`, `Table.Head`, `Table.Body`, `Table.Row`, `Table.HeaderCell`, `Table.Cell`
- Table: `w-full text-sm text-left` in a `overflow-x-auto` wrapper div
- Head: `bg-surface-50 border-b border-surface-200`
- HeaderCell: `px-4 py-3 text-label font-semibold text-surface-600 uppercase tracking-wider text-xs`
- Row: `border-b border-surface-100 hover:bg-surface-50 transition-colors`
- Cell: `px-4 py-3 text-surface-700`
- Support `striped` prop on Table.Body for alternating row colors

**StatCard.tsx:**
- Props: `title` (string), `value` (string | number), `change` ({value: number, direction: 'up' | 'down' | 'flat'}), `icon` (ReactNode), `prefix` (string, optional — e.g., 'R' for currency), `loading` (boolean)
- Layout: Card with icon top-right, title (text-caption), value (text-2xl font-bold), change indicator at bottom
- Change: up = green text + arrow up, down = red text + arrow down, flat = gray text + dash
- Change displays as percentage: `+12.5%` or `-3.2%`
- Loading state: show Spinner in place of value
- Use `surface-card` base class, padding md

**Tabs.tsx:**
- Props: `tabs` ({id: string, label: string, icon?: ReactNode}[]), `activeTab` (string), `onChange` ((id: string) => void)
- Horizontal tab bar with underline active indicator
- Active: `text-primary-600 border-b-2 border-primary-600`
- Inactive: `text-surface-500 hover:text-surface-700`
- All: `px-4 py-3 text-sm font-medium transition-default`
- Support `variant` prop: 'underline' (default) | 'pills' (rounded pill style tabs)
