- src/index.css with CSS custom properties for semantic tokens, base layer resets, and utility classes

</objective>

<task type="auto">
  <name>Task 1: Expand Tailwind config with full design system tokens</name>
  <files>tailwind.config.js</files>
  <action>
Rewrite tailwind.config.js with the complete FableDash design system. The existing config already has primary (indigo-blue), secondary (slate-blue), and accent (warm orange) palettes — refine and expand them:

**Colors:**
- `primary` — Keep the indigo-blue family (50-950). This is Fable's core brand. Used for navigation, CTAs, active states. Refine the existing scale for better contrast ratios.
- `secondary` — Keep the cool slate-blue (50-950). Used for secondary actions, muted elements.
- `accent` — Keep the warm amber/orange (50-950). Used for highlights, notifications, creative energy.
- `success` — Green scale (50-950) for positive states, revenue up, healthy metrics.
- `warning` — Amber/yellow scale (50-950) for caution states, approaching limits.
- `danger` — Red scale (50-950) for errors, negative metrics, alerts.
- `surface` — Neutral gray scale tuned warm-ish (not blue-cold). Used for backgrounds, cards, borders. Name the scale: `surface-50` through `surface-950`.

**Typography:**
- Extend `fontFamily` with `sans: ['Inter', ...defaultTheme.fontFamily.sans]` (Inter is the primary font — clean, professional, excellent readability)
- Extend `fontSize` with a named scale: `xs` (0.75rem), `sm` (0.875rem), `base` (1rem), `lg` (1.125rem), `xl` (1.25rem), `2xl` (1.5rem), `3xl` (1.875rem), `4xl` (2.25rem), `display` (3rem) — each with matching line-height
- Do NOT remove Tailwind's default fontSize values; only extend with `display`

**Spacing:**
- Add named spacing tokens in `extend.spacing`: `4.5` (1.125rem), `18` (4.5rem), `88` (22rem) for sidebar width
- These supplement Tailwind defaults, not replace

**Border Radius:**
- Keep existing xl/2xl/3xl. Add `DEFAULT: '0.5rem'` for a slightly softer default feel (not sharp corners — warm/approachable).

**Shadows:**
- Keep `card` and `card-hover`. Add:
  - `soft`: `0 2px 8px rgba(0,0,0,0.04)` — subtle elevation for inputs/badges
  - `medium`: `0 4px 12px rgba(0,0,0,0.08)` — standard card elevation
  - `strong`: `0 8px 24px rgba(0,0,0,0.12)` — modals, dropdowns
  - `glow-primary`: `0 0 20px rgba(101,113,245,0.15)` — primary focus ring
  - `inner-soft`: `inset 0 2px 4px rgba(0,0,0,0.04)` — input pressed states

**Background Image:**
- Keep `gradient-primary`. Add:
  - `gradient-accent`: warm orange gradient for creative/highlight sections
  - `gradient-dark`: dark surface gradient for sidebar or hero areas

**Animation (extend.animation + keyframes):**
- `fade-in`: opacity 0->1 over 200ms ease-out
- `slide-up`: translateY(8px)->0 + opacity 0->1 over 300ms ease-out
- `pulse-soft`: subtle opacity pulse (1->0.7->1 over 2s infinite) for loading states

Import `defaultTheme` from `tailwindcss/defaultTheme` at the top of the config.

Do NOT install any Tailwind plugins. Keep `plugins: []`.
