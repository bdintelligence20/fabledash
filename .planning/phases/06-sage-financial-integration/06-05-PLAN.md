       - router = APIRouter()
       - All endpoints require Depends(get_current_user)

       GET /summary — Aggregated financial overview:
       - Query params: period (str, optional — e.g., "2026-01" or "2026-Q1" or "2026")
       - Logic:
         1. Get latest financial snapshot from Firestore (most recent by period_end)
         2. Get invoice stats: count by status, total outstanding (unpaid/overdue), total paid
         3. Get latest P&L upload for the period (if any)
         4. Get latest revenue forecast (if any)
         5. Calculate: revenue_this_month, revenue_last_month, revenue_growth_rate
       - Return aggregated response:
         ```json
         {
           "success": true,
           "data": {
             "snapshot": { latest FinancialSnapshot or null },
             "invoices": {
               "total": count,
               "paid": count,
               "outstanding": count,
               "overdue": count,
               "total_revenue": float,
               "total_outstanding_amount": float
             },
             "pnl": { latest P&L summary or null },
             "forecast": { latest forecast summary or null },
             "sage_connected": bool
           }
         }
         ```

       GET /revenue-trend — Monthly revenue trend:
       - Query params: months (int, default 6)
       - Query financial snapshots for last N months
       - Return: { success: true, data: [{ period, revenue, expenses, net_profit }] }

    2. Update python-backend/app/main.py:
       - Import and include financial_data_router with prefix="/financial-data", tags=["financial-data"]
