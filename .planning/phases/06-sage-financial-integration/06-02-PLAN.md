  <action>
    Create python-backend/app/utils/sage_sync.py — service that syncs data from Sage API to Firestore:

    Class SageSyncService:
    - __init__(self, sage_client: SageClient): stores client reference

    - async sync_invoices(self, since: date | None = None) -> dict:
      1. GET /sales_invoices from Sage (with date_from filter if since provided)
      2. For each invoice from Sage:
         - Map Sage fields to InvoiceResponse model (sage_id, invoice_number, status, amount, dates)
         - Try to match client by comparing Sage contact name to Firestore client names (best-effort linking)
         - Upsert to Firestore invoices collection (use sage_id as document ID for idempotent sync)
      3. Return: { synced: count, errors: count }

    - async sync_payments(self, since: date | None = None) -> dict:
      1. GET /contact_payments from Sage (with date_from filter if since provided)
      2. Map each payment to PaymentResponse model
      3. Link to invoices by matching Sage payment.invoice references
      4. Upsert to Firestore payments collection
      5. Return: { synced: count, errors: count }

    - async sync_balances(self) -> dict:
      1. GET /ledger_accounts from Sage to get account balances
      2. Extract: cash_on_hand (bank accounts), accounts_receivable, accounts_payable
      3. Return balance summary dict (not stored directly — used in snapshot)

    - async create_snapshot(self, period_start: date, period_end: date) -> FinancialSnapshot:
      1. Query invoices and payments within the period from Firestore
      2. Calculate: total_revenue (sum of paid invoices), total_expenses (from Sage ledger if available, else 0)
      3. net_profit = total_revenue - total_expenses
      4. Get current balances via sync_balances()
      5. Create FinancialSnapshot document in Firestore
      6. Return snapshot

    - async full_sync(self) -> dict:
      1. Call sync_invoices() — full sync (no date filter)
      2. Call sync_payments() — full sync
      3. Create snapshot for current month
      4. Return combined results: { invoices: {...}, payments: {...}, snapshot_id: str }

    Handle errors gracefully — if Sage isn't connected, return error dict instead of raising.
    Log all sync operations with counts and timing.
