<task type="auto">
  <name>Task 1: Create auth context and hook</name>
  <files>src/contexts/AuthContext.tsx, src/hooks/useAuth.ts</files>
  <action>
    1. Create src/contexts/AuthContext.tsx:
       - Import { auth } from '../lib/firebase'
       - Import { onAuthStateChanged, signInWithEmailAndPassword, signOut, User } from 'firebase/auth'
       - Define AuthContextType interface: { user: User | null, loading: boolean, error: string | null, login: (email: string, password: string) => Promise<void>, logout: () => Promise<void>, getIdToken: () => Promise<string | null> }
       - Create AuthContext with createContext<AuthContextType>
       - Create AuthProvider component:
         a. useState for user (User | null), loading (boolean, default true), error (string | null)
         b. useEffect with onAuthStateChanged listener that sets user and loading=false
         c. login function: signInWithEmailAndPassword, catch errors and set error state. Map Firebase error codes to user-friendly messages (auth/invalid-credential → "Invalid email or password", auth/too-many-requests → "Too many attempts, try again later")
         d. logout function: signOut(auth)
         e. getIdToken function: return auth.currentUser?.getIdToken() ?? null
         f. Clean up onAuthStateChanged listener on unmount
       - Export AuthProvider and AuthContext

    2. Create src/hooks/useAuth.ts:
       - Import { useContext } from 'react'
       - Import { AuthContext } from '../contexts/AuthContext'
       - Export useAuth hook that calls useContext(AuthContext) and throws if used outside AuthProvider
  </action>
  <verify>npx tsc --noEmit 2>&1 | head -20</verify>
  <done>AuthContext with Firebase auth state management. useAuth hook for consuming auth state. Login/logout/getIdToken functions. Error handling with user-friendly messages.</done>
</task>
